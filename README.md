# MyBike page

This is a static web page project built with HTML, CSS and Sass. It presents the store that sells new generation bicycles.

The design of this project is based on [this reference](https://www.figma.com/file/NZQAIydtHo5QkINyGLHNcq/BIKE-New-Version?node-id=0%3A1).

### Tech stack

- HTML
- CSS (SCSS)
- Sass

### Getting started

If you want to run the project locally, do the following steps:

1. Clone the repository:
```
git clone https://github.com/dminikulin/mybike-page.git
```
2. Install dependencies:
```
npm install
# or
yarn install
```
3. Run the project in local environment:
```
npm install
# or
yarn install
```
